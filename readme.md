🌤️ weather-app

## 📝 Descripción
Una aplicación de línea de comandos (CLI) para consultar el clima actual en cualquier ciudad del mundo.

## ✨ Características Principales
- Búsqueda por ciudad
- Geocodificación automática
- Clima actual detallado
- Caché local (1 hora)
- Salida formateada por consola

## 🚀 Tecnologías Utilizadas
- Node.js
- API de Open-Meteo
- Fetch Nativo (Web API)

## 🛠️ Pasos para comenzar a usar la aplicación: 
-Clona el repositorio
-cd weather-app
-node index.js "NombreCiudad"

## 💻 Uso
Puedes ejecutar la aplicación pasando el nombre de la ciudad como argumento:
npm start "Madrid"
node src/index.js "Buenos Aires"
